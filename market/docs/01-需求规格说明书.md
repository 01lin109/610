# 商城后台管理系统 - 需求规格说明书

## 1. 项目概述

### 1.1 项目名称
商城后台管理系统（Market Admin）

### 1.2 项目背景
随着电商业务规模扩大，需要一个功能完备的后台管理系统来实现商品、订单、会员、促销、财务等核心业务的统一管理。系统需支持多角色权限控制，提供数据可视化统计分析，帮助运营团队高效决策。

### 1.3 目标用户
- 系统管理员：负责系统配置、用户权限管理
- 运营人员：负责商品上下架、促销活动配置、内容发布
- 财务人员：负责交易流水查看、退款审核
- 客服人员：负责订单处理和会员查询

## 2. 功能需求

### 2.1 商品管理模块
| 功能 | 描述 |
|------|------|
| 商品列表 | 分页查询、关键词搜索、分类筛选、状态筛选 |
| 商品增删改 | 新增商品（名称/编码/分类/价格/库存等）、编辑、删除 |
| 商品上下架 | 一键上下架切换 |
| 分类管理 | 多级分类树管理、排序、启用/禁用 |

### 2.2 订单管理模块
| 功能 | 描述 |
|------|------|
| 订单列表 | 分页查询、订单号搜索、状态筛选 |
| 订单详情 | 查看订单基本信息、商品明细、收货信息 |
| 订单状态流转 | 待付款→已支付→已发货→已完成，支持手动关闭 |
| 发货操作 | 已支付订单支持一键发货 |

### 2.3 会员管理模块
| 功能 | 描述 |
|------|------|
| 会员列表 | 分页查询、用户名/手机号搜索、状态筛选 |
| 会员编辑 | 编辑昵称、手机号、积分、余额等信息 |
| 会员启用/禁用 | 状态切换 |
| 会员等级 | 等级名称、折扣率、升级条件配置 |

### 2.4 促销管理模块
| 功能 | 描述 |
|------|------|
| 优惠券管理 | 满减券/折扣券/代金券的创建、编辑、删除 |
| 活动管理 | 满减活动、限时折扣、秒杀、拼团的创建和管理 |
| 活动时间设置 | 设定活动开始/结束时间，自动切换活动状态 |

### 2.5 运营管理模块
| 功能 | 描述 |
|------|------|
| 轮播图管理 | 首页轮播图的增删改查、排序、启用/禁用 |
| 广告管理 | 图片/链接/视频广告的创建和管理，支持指定广告位 |

### 2.6 内容管理模块
| 功能 | 描述 |
|------|------|
| 文章管理 | 帮助文档、资讯文章的发布、编辑、置顶 |
| 公告管理 | 系统公告、活动通知的发布和管理 |

### 2.7 统计报表模块
| 功能 | 描述 |
|------|------|
| 销售概览 | 总商品数、会员数、订单状态分布统计 |
| 销售趋势 | 按日统计销售额趋势图（ECharts） |
| 商品销量排行 | Top N 商品销量统计 |

### 2.8 财务管理模块
| 功能 | 描述 |
|------|------|
| 交易流水 | 支付/退款/充值的流水记录查询 |
| 退款管理 | 退款申请的列表、详情、审核通过/驳回 |

### 2.9 权限管理模块
| 功能 | 描述 |
|------|------|
| 用户管理 | 系统用户的 CRUD、状态切换 |
| 角色管理 | 角色的 CRUD、状态管理 |
| 菜单管理 | 系统菜单树的 CRUD、排序 |
| 部门管理 | 组织架构部门的 CRUD、负责人配置 |

### 2.10 系统设置模块
| 功能 | 描述 |
|------|------|
| 基本设置 | 站点名称、版权信息、订单超时时间 |
| 物流设置 | 运费配置、包邮门槛 |
| 支付设置 | 微信支付/支付宝开关 |
| 短信设置 | 短信服务商、签名配置 |

## 3. 非功能需求

### 3.1 性能需求
- 页面响应时间 ≤ 2 秒
- 支持 100+ 并发用户
- 数据库查询响应 ≤ 500ms

### 3.2 安全需求
- 基于 JWT 的无状态认证
- 密码 BCrypt 加密存储
- 接口权限控制（角色-菜单关联）
- CORS 跨域安全配置

### 3.3 可用性需求
- 系统 7×24 小时可用
- 数据库定期备份
- 支持 Docker 容器化部署

## 4. 技术栈

| 层级 | 技术选型 |
|------|----------|
| 后端框架 | Spring Boot 2.x |
| 安全认证 | Spring Security + JWT |
| 持久层 | MyBatis + MySQL |
| 缓存 | Redis |
| 前端框架 | Vue 3 + Vite |
| UI 组件库 | Element Plus |
| 图表 | ECharts 5 |
| 状态管理 | Pinia |
| HTTP 客户端 | Axios |
| 部署 | Docker + Nginx |
