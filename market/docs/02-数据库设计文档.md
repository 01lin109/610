# 商城后台管理系统 - 数据库设计文档

## 1. 数据库概述

- 数据库名称: `market_db`
- 字符集: utf8mb4
- 排序规则: utf8mb4_general_ci
- 存储引擎: InnoDB

## 2. 表结构设计

## 4. ER 图概要

```
sys_dept ──1:N── sys_user ──N:N── sys_role ──N:N── sys_menu
product_category ──1:N── product ──N:N── `order` ──1:N── order_item
member ──N:1── member_level
member ──1:N── `order`
member ──1:N── transaction
`order` ──1:N── refund
`order` ──1:N── transaction
```

## 5. 索引设计

- 主键索引：所有表 id 字段
- 唯一索引：sys_user.username, order.order_no, refund.refund_no
- 普通索引：order.member_id, order.status, product.category_id, transaction.order_id
- 联合索引：(member_id, create_time) 用于会员订单查询
