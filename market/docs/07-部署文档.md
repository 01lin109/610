# 商城后台管理系统 - 部署文档

## 1. 环境要求

### 1.1 生产环境
- 操作系统：Linux (CentOS 7+ / Ubuntu 20.04+) 或 Windows Server
- Docker: 20.10+
- Docker Compose: 2.0+
- 内存：≥ 4GB
- 磁盘：≥ 20GB

### 1.2 开发环境
- JDK 8+
- Maven 3.6+
- Node.js 16+
- MySQL 8.0
- Redis 7

## 2. Docker 部署（推荐）

### 2.1 目录结构
```
market/
├── market_backend/
│   ├── Dockerfile
│   ├── docker-compose.yml
│   ├── target/market_backend.jar
│   └── sql/init.sql
└── market_frontend/
    └── dist/
```

### 2.2 构建后端 JAR
```bash
cd market_backend
mvn clean package -DskipTests
```

### 2.3 构建前端
```bash
cd market_frontend
npm install
npm run build
```

### 2.4 启动服务
```bash
cd market_backend
docker-compose up -d
```

### 2.5 查看状态
```bash
docker-compose ps
docker-compose logs -f backend
```

### 2.6 停止服务
```bash
docker-compose down
```

### 2.7 数据持久化
MySQL 数据卷：`mysql_data`
Redis 数据卷：`redis_data`

```bash
# 备份数据库
docker exec market-mysql mysqldump -uroot -proot123 market_db > backup.sql
```

## 3. 手动部署

### 3.1 数据库初始化
```bash
mysql -u root -p < sql/init.sql
```

### 3.2 启动后端
```bash
cd market_backend
mvn spring-boot:run
# 或
java -jar target/market_backend.jar
```

### 3.3 启动前端
```bash
cd market_frontend
npm run dev
```

## 4. Nginx 配置

```nginx
server {
    listen 80;
    server_name localhost;

    # 前端静态资源
    location / {
        root /usr/share/nginx/html;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # API 反向代理
    location /api/ {
        proxy_pass http://backend:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
```

## 5. 环境变量

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| MYSQL_ROOT_PASSWORD | root123 | MySQL root密码 |
| SPRING_DATASOURCE_URL | jdbc:mysql://mysql:3306/market_db | 数据库连接 |
| SPRING_REDIS_HOST | redis | Redis主机 |
| JWT_SECRET | market-secret-key-2026 | JWT密钥 |

## 6. 健康检查

| 服务 | 地址 | 预期 |
|------|------|------|
| 后端 | http://localhost:8080/actuator/health | {"status":"UP"} |
| MySQL | localhost:3306 | 连接成功 |
| Redis | localhost:6379 | PONG |

## 7. 常见问题

### Q1: 端口冲突
```bash
# 检查端口占用
netstat -ano | findstr 8080
# 修改 docker-compose.yml 端口映射
```

### Q2: MySQL 连接失败
- 检查 docker-compose.yml 中 MYSQL_ROOT_PASSWORD
- 检查 application.yml 中数据库配置是否匹配

### Q3: 前端页面空白
- 检查 Nginx try_files 配置
- 确认前端路由模式为 hash 或已配置 history 模式回退
